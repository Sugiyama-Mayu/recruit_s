作品の詳細
■作品1 プラトレ
 ・ゲーム概要
  3Dパズルゲームです。
  イルミネーション(線路)を繋げて、電車を走らせ、人を乗せていき、最後の星を目指すゲームです。
  
  2年生の時の作品です。
  星(マップ)の複合回転に合わせたオブジェクトの動きや、、線路の配置位置の調整、人のランダム配置(詰まないように位置を調整)をしており、
  外部ファイルにデータを出力して、ランキング用のスコアの保持や、セーブデータを残すようにしています。
  軽量化のために 線路のエフェクトを正面(線路を配置するマップ面)または、プレイキャラがいる面でプレイキャラから一定距離以内に線路がある場合、発生させています。
  二画面表示はビューポートの設定で実現しています。
  
■作品2 Sweets Manufacturer
 ・ゲーム概要
  3Dクッキングシミュレーションゲームです。
  ベルトコンベア上の材料をクリックし、ミニゲームを成功させお菓子を完成させていくゲームです。
  
  3年生の時の作品です。
  グループ制作でリーダー兼メインプログラマを担い、プログラム全般を一人で担当しました。
  リアルなクッキングを表現するために12種類のミニゲーム、コレクション要素やハードモードのやりこみ要素も実装しました。
  取得アイテムのデータは外部ファイルに出力して保存しています。
  
■作品3 栗デレラ
 ・ゲーム概要
  アクションゲームです。
  お正月の前日の夜のこと、
  おせちになることに遅れてしまったイガグリが、次の日までに急いで栗きんとんになるというゲームです。
  
  1年生の時の作品です。
  敵や武器の強化ステージをランダムに出現させ、特定の敵に触れるとプレイキャラの姿が変わるようにし、
  外部ファイルを使って、スコアのランキングを表示されるようにしました。
  敵が画面外でも弾を打っていたので、画面外で弾を打たない処理を施しました。
    
  
※作品についてvisualstudioが入っていないPCでも確認済みですが、
  VisualStudioでコンパイルして動作させるにはDirectX SDKが必要になります。
